# PySpark Customer Sales Analysis & Segmentation

## 📌 Project Overview
We performed customer and sales data analysis using PySpark. It includes data aggregation, customer insights, and segmentation into Gold, Silver, and Bronze categories.



## 📂 Dataset
- customers.csv → Contains customer details  
- sales.csv → Contains sales transaction data  

---

## 🚀 Features

### 1. Daily Sales Analysis
Calculates total sales for each day to understand revenue trends.

### 2. City-wise Revenue
Computes total revenue generated from each city.

### 3. Top 5 Customers
Identifies top customers based on their total spending.

### 4. Repeat Customers
Finds customers who made more than one purchase.

### 5. Customer Segmentation
Customers are categorized based on total spending:
- Gold → total_spend > 10000  
- Silver → 5000 to 10000  
- Bronze → total_spend < 5000  

### 6. Final Reporting Table
Combines all important insights into a single table including:
- Customer Name  
- City  
- Total Spend  
- Order Count  
- Segment  

---

##  Logic 
Customer segmentation helps businesses:
- Identify high-value customers  
- Improve targeted marketing strategies  
- Make better business decisions

## Output Analysis
- Daily revenue trends  
- City-wise performance
